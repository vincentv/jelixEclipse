Les formulaires dans Jelix

Jelix propose jForms, un système permettant de générer des formulaires mais aussi d’en gérer les données. Vous pouvez aussi créer et gérer vos formulaires de manières plus classique, sans jForms (qui n’est pas forcément adaptés à toutes les situations). Les deux façons sont décrites dans les deux rubriques suivantes :

  • formulaires classiques
  • Utilisation de jForms

Documentation générée à partir de la documentation du site Jelix avec JbManuel pour le projet JelixEclipse
