Configuration requise

Vous devez installer au minimum la version 5.0 de PHP. Les extensions dom, simplexml, pcre, session, spl sont requises (elles sont actives en général dans une installation standard de PHP 5.0).

Il faut aussi l’extension tokenizer. Cette extension n’est pas toujours installée. Avec certaines distributions linux (comme ubuntu), cette extension tokenizer n’est pas livrée avec le paquet PHP5. Veillez donc à installer le paquet correspondant.

Vous pouvez installer aussi un connecteur de base de données. Pour le moment Jelix prend en charge mysql 4.0+, postgresql 8.0+, sqlite et pdo.

Si vous comptez utiliser les scripts en ligne de commande d’aide au développement (jelix-scripts), il vous faut aussi installer la version ligne de commande de PHP : PHP-CLI.

Configuration de php

  • magic_quotes_gpc et magic_quotes_runtime doivent être à Off. Si ce n’est pas le cas, il faut activer le plugin magicquotes livré avec jelix.
  • session.auto_start doit être à 0
  • safe_mode doit être à Off (jelix n’a pas été testé à on)

Il est également recommandé de mettre ces valeurs :

  • register_globals = off
  • asp_tags = off
  • short_open_tag = off

Un phpinfo() peut vous permettre de vérifier votre configuration. Par défaut la configuration se trouve dans un fichier nommé php.ini.
